# 📚 CCNA2 – Site de Révision et de Quiz

> Un site web interactif dédié à la révision et à l'entraînement par quiz pour la certification **Cisco CCNA 2 (Switching, Routing, and Wireless Essentials)**.

---

## 🎯 Objectif

Ce projet a pour but d'aider les étudiants et les candidats à la certification CCNA 2 à :

- **Réviser** les notions théoriques des différents modules du cours CCNA 2.
- **S'entraîner** grâce à des quiz interactifs sous forme de questions à choix multiples (QCM).
- **Évaluer** leur niveau de compréhension sur chaque thème abordé.
- **Progresser** à leur propre rythme, module par module.

---

## 📖 Contenu couvert

Le site couvre les principaux thèmes du programme CCNA 2 (*Switching, Routing, and Wireless Essentials*), notamment :

| Module | Thème |
|--------|-------|
| Module 1 | Concepts de base des réseaux commutés |
| Module 2 | Configuration des commutateurs (VLANs, STP) |
| Module 3 | Routage inter-VLAN |
| Module 4 | Concepts du routage |
| Module 5 | Routage statique |
| Module 6 | Routage dynamique (OSPF) |
| Module 7 | Listes de contrôle d'accès (ACL) |
| Module 8 | NAT pour IPv4 |
| Module 9 | Concepts des réseaux sans fil (Wireless) |
| Module 10 | Configuration WLAN |
| Module 11 | Sécurité du réseau et concepts avancés |

---

## 🧩 Fonctionnalités

- ✅ **Quiz par module** : des questions spécifiques à chaque chapitre du cours.
- ✅ **Questions à choix multiples (QCM)** : format proche des examens officiels Cisco.
- ✅ **Correction instantanée** : la bonne réponse est affichée après chaque question.
- ✅ **Score final** : résumé du résultat à la fin de chaque quiz.
- ✅ **Interface intuitive** : navigation simple et claire.
- ✅ **Fiches de révision** : résumés des concepts-clés par leçon.

## 🚀 Démarrage rapide

### Prérequis

Aucun prérequis particulier. Il suffit d'un navigateur web moderne (Chrome, Firefox, Edge, etc.).

### Lancer le projet en local

1. Cloner le dépôt :

   ```bash
   git clone https://github.com/7ShIkI3/CCNA2-Quizz-Lesson.git
   ```

2. Accéder au dossier du projet :

   ```bash
   cd CCNA2-Quizz-Lesson
   ```

3. Ouvrir le fichier `index.html` dans votre navigateur.
